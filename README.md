# Toumï Solutions: Concevoir et déployer une application complète de gestion des tâches collaboratives

## Description
Cet application web et mobile complète permettant la gestion des tâches en équipe : création, assignation, suivi d’avancement et statistiques.

### Installation

### Installer l’interface CLI d’Ionic globalement
npm install -g @ionic/cli
### Initialisation du projet avec Ionic + Angular
ionic start nom de app --type=angular
### exécuter le programme
ionic serve
### Technologies et outils utilisées
- Ionic+Framework
- Github
- Javascript
- Figma
